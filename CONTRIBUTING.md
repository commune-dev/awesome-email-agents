# Contributing to awesome-email-agents

Thank you for contributing!

## How to add an entry

1. Fork this repo
2. Add your tool/resource to the relevant section in README.md
3. Follow the existing format: **Name** — one-sentence description, then bullets with install/links
4. Open a pull request with a brief description of what you're adding

## Criteria

- **Purpose-built for agents** or proven production adoption in agent stacks
- Actively maintained (last commit within 12 months for code repos)
- Not spam or self-promotion without substantial content

## Format

```
**[Tool Name](https://link)** — One-sentence description of what it does and why it's relevant for agents.
- Install: `pip install package`
- GitHub: [org/repo](https://github.com/org/repo)
- Purpose-built / Dual-use
```

## What we don't accept

- General-purpose email libraries without agent-specific features or adoption
- Abandoned or unmaintained tools
- Duplicates
